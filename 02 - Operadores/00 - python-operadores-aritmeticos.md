## Suma (`+`)
```python
a = 5
b = 3
resultado = a + b  # resultado es 8
```


## Resta (`-`)
```python
a = 10
b = 7
resultado = a - b  # resultado es 3
```


## Multiplicación (`*`)
```python
a = 5
b = 4
resultado = a * b  # resultado es 20
```


## División (`/`)
```python
a = 10
b = 2
resultado = a / b  # resultado es 5.0
```


## División Entera (`//`)
```python
a = 10
b = 3
resultado = a // b  # resultado es 3
```


## Módulo (`%`)
```python
a = 10
b = 3
resultado = a % b  # resultado es 1
```


## Potencia (`**`)
```python
a = 2
b = 3
resultado = a ** b  # resultado es 8
```
