## Booleanos (`bool`)
```python
verdadero = True
falso = False
```


## Números enteros (`int`)
```python
numero_entero = 10
```


## Números de punto flotante (`float`)
```python
numero_flotante = 3.14
```


## Cadenas de texto (`str`)
```python
cadena_texto = "Hola, mundo!"
```


## Listas (`list`)
```python
mi_lista = [1, 2, 3, 4, 5]
```


## Tuplas (`tuple`)
```python
mi_tupla = (10, 20, 30, 40, 50)
```


## Diccionarios (`dict`)
```python
mi_diccionario = {"nombre": "Juan", "edad": 30, "ciudad": "Madrid"}
```


## Conjuntos (`set`)
```python
mi_conjunto = {1, 2, 3, 4, 5}
```
