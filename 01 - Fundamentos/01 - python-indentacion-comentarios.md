## Ejemplo de indentación en Python
```python
# Declaración de variables
x = 5
y = 10

# Definición de función
def suma(a, b):
    # Bloque de código indentado
    resultado = a + b
    return resultado

# Estructura de control
if x < y:
    # Bloque de código indentado
    print("x es menor que y")
else:
    # Otro bloque de código indentado
    print("y es menor que x")
```
